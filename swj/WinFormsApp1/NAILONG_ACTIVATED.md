## ??? 如果图片没有显示

### 检查清单

1. **确认文件存在**
   ```
   ? bin\Debug\net8.0-windows\nailong.png 存在吗？
   ```

2. **检查调试输出**
   - 打开 Visual Studio → 菜单 → 调试 → 窗口 → 输出
   - 查看是否有 `? 成功加载背景图片！` 的信息

3. **如果文件不在 bin 文件夹**
   - 方案 A：把文件复制到 `bin\Debug\net8.0-windows\` 文件夹
   - 方案 B：在 Visual Studio 项目中右键 → 添加 → 现有项 → 选择 `nailong.png`
     - 然后右键 → 属性 → "复制到输出目录" 改为 "如果较新则复制"

4. **检查文件格式**
   - 确保文件确实是 PNG 格式（不是快捷方式）
   - 可以尝试用图片查看器打开验证

## ?? 自定义遮罩效果

如果觉得遮罩太深或太浅，可以调整透明度：

```csharp
// 在 SetPanelBackground() 方法中找到这行
Color.FromArgb(60, UIHelper.Colors.Primary)
//                  ↑ 改这个数字

// 数字范围：0-255
// 0    = 完全透明（看不到遮罩，只有原图）
// 60   = 中等透明（现在的设置）
// 100  = 较深（遮罩更明显，背景更暗）
// 200  = 很深（遮罩很明显，背景很暗）
// 255  = 完全不透明（只看到蓝色，看不到背景）

// 建议值：40-80 之间，取决于你的图片亮度
```
