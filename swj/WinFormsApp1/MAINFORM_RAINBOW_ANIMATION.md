# ?? 主窗口彩虹渐变背景动画

## 功能概述

为主窗口的顶部导航栏添加**流动的彩虹渐变背景动画**，营造温暖、炫彩、高端的视觉效果，类似奶龙背景的温馨感。

## ?? 色彩设计

### 彩虹色系（7 色循环）
```
红 → 橙 → 黄 → 绿 → 青 → 蓝 → 紫 → 红 (循环)
```

| 颜色 | RGB | 位置 |
|------|-----|------|
| 红 | (255, 0, 0) | 0% |
| 橙 | (255, 127, 0) | 16.7% |

| 蓝 | (0, 0, 255) | 83.3% |
| 紫 | (139, 0, 255) | 100% |

## ?? 动画效果

### 1?? 色相旋转
- **旋转角度**：每 50ms 增加 2°
- **完整循环**：360° ÷ 2° = 180 × 50ms ≈ **9 秒完成一圈**
- **效果**：彩虹色从左到右持续流动，营造流水般的感觉

### 2?? 白色半透明叠加
- **透明度**：80（透明度 31%）
- **作用**：让彩虹背景更柔和，保证文字可读性
- **视觉效果**：温暖而不刺眼

### 3?? 装饰光点
- **数量**：6 个漂浮光点
- **大小**：15±8px（动态缩放）
- **透明度**：100±80（随时间变化）
- **位置**：均匀分布在顶部
- **效果**：呼吸式的柔和闪烁，增强活力

## ?? 色彩渐变过程

```
时间→
0°   : 红橙黄 ─→ 绿 ─→ 青蓝紫
2°   : 橙黄绿 ─→ 青 ─→ 蓝紫红
4°   : 黄绿青 ─→ 蓝 ─→ 紫红橙
...
360° : 回到红色 (循环)
```

每一帧渐变色都在左→右流动，整体效果就像**彩虹在缓缓滚动**。

## ?? 代码结构

### 初始化
```csharp
private void MainForm_Load(object sender, EventArgs e)
{
    UIHelper.StyleForm(this);
    StyleButtons();
    StartRainbowAnimation();  // ← 启动动画
}
```

### 启动动画
```csharp
private void StartRainbowAnimation()
{
    animationTimer = new System.Windows.Forms.Timer();
    animationTimer.Interval = 50;  // 50ms 一帧
    animationTimer.Tick += (s, e) =>
    {
        hueRotation += 2f;  // 每帧增加 2°
        if (hueRotation >= 360f) hueRotation = 0f;
        pnlHeader.Invalidate();  // 触发重绘
    };
    animationTimer.Start();
    pnlHeader.Paint += PnlHeader_Paint;  // 自定义绘制
}
```

### 绘制渐变
```csharp
private void PnlHeader_Paint(object sender, PaintEventArgs e)
{
    // 1. 创建 7 点彩虹渐变
    LinearGradientBrush brush = new LinearGradientBrush(...);
    ColorBlend colorBlend = new ColorBlend(7);
    colorBlend.Colors = new[] { 红, 橙, 黄, 绿, 青, 蓝, 紫 };
    brush.InterpolationColors = colorBlend;
    
    // 2. 填充背景
    e.Graphics.FillRectangle(brush, rect);
    
    // 3. 叠加半透明白色（保证文字清晰）
    e.Graphics.FillRectangle(overlayBrush, rect);
    
    // 4. 绘制装饰光点
    DrawLightParticles(e.Graphics, rect);
}
```
